# Como Implementar o Ecossistema de 50 Skills no Claude

## Pré-requisitos

- Conta no claude.ai (plana Pro ou Team)
- Acesso ao recurso Projects

## Passo a Passo

### 1. Crie um Project no Claude

No menu lateral do claude.ai, clique em Projects > + New Project.
Nomeie como "Vendas B2B" ou "Protocolo Hidra".

### 2. Abra as Instruções do Project

Clique em "Edit project details".
Role até "Project instructions".

### 3. Cole as Skills

Copie o código de cada skill e cole no campo de instruções.
Você pode empilhar múltiplas skills no mesmo Project.

### 4. Ative com Gatilhos

Em qualquer conversa dentro do Project, use as frases de gatilho:

- "write a cold email to [name] at [company]"
- "map this deal to MEDDIC"
- "build a close plan for [deal name]"

O Claude executa o fluxo completo automaticamente.
